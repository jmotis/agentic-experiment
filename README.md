# Gaming the Great Plague

An interactive fiction game built with [Twine 2](https://twinery.org/) and the [SugarCube 2](https://www.motoslave.net/sugarcube/2/) story format.

## About

Gaming the Great Plague is a browser-based interactive fiction experience that explores themes related to the Great Plague. Players navigate through the story by making choices that shape the narrative.

## How to Play

1. Download or clone this repository
2. Open `GamingtheGreatPlague.html` in any modern web browser (Chrome, Firefox, Safari, Edge)

No installation, server, or internet connection is required — the game is entirely self-contained in a single HTML file.

## Development

This game is authored using the [Twine 2](https://twinery.org/) editor. To make changes:

1. Open Twine 2 (desktop app or [online editor](https://twinery.org/))
2. Import `GamingtheGreatPlague.html` into Twine
3. Edit passages and story logic within the Twine editor
4. Publish/export the updated HTML file back to the repository

## Technology

- **Twine 2** — Interactive fiction authoring tool
- **SugarCube 2.37.3** — Story format providing the game engine, macro system, and save/load functionality
